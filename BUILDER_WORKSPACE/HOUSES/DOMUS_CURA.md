### Organism Wellness Score

Let `n_i` = level of neurochemical channel `i`, i ∈ {dopamine, serotonin, norepinephrine, cortisol, oxytocin, GABA, glutamate, acetylcholine}  
Let `rank_i` = channel sovereign rank (1 through 8, φ-assigned by physiological priority)  
Let `R` = Kuramoto coherence coefficient across all 43 active cores  
Let `φ` = 1.6180339887

```
Wellness Score:
  W = [Σ(n_i × φ^rank_i) for i=1..8] / 8 × R

  W ≥ φ⁻¹ = 0.618  →  FUNCTIONAL (organism operating, care monitoring active)
  W ≥ 0.95          →  THRIVING   (organism superadient, all channels aligned)
  φ⁻² ≤ W < φ⁻¹     →  CARE REQUIRED (one or more channels below functional threshold)
  W < φ⁻²  = 0.382  →  CARE EMERGENCY (multiple channels depleted, anti-collapse engaged)

```
